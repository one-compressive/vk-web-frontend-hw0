(generator) fibonacciGenerator(n) → {object}

Функция, которая принимает одно число n и возвращает генератор, генерирующий первые n чисел последовательности Фибоначчи. Если n меньше или равно 0, генератор не должен генерировать никаких значений.

n Number

// 1, затем 2, затем 3
const gen = fibonacciGenerator(3);

  alert(value);
